# 🇩🇪 German Analytics Job Market Analysis

## Project Overview

This is an end-to-end data analytics project exploring the German job market for Analytics positions.

The project collects job postings, transforms and stores the data in PostgreSQL, performs SQL analysis and visualizes insights using Power BI.

---

## Dataset

The project uses publicly available job postings collected from major German job platforms.

Planned sources:

- StepStone
- LinkedIn Jobs
- Indeed Germany
- Xing Jobs (optional)
- Glassdoor (optional)

---

## Business Questions

This project answers the following questions:

- Which Analytics positions are most frequently advertised?
- Which skills are most requested by employers?
- Which cities have the highest demand for Analysts?
- Which companies are hiring the most?
- Which BI tools are requested most often?
- What salary ranges are offered?
- How common are remote, hybrid and on-site positions?

## Project Pipeline

1. Collect job postings
2. Clean and transform data

4. Analyze data using SQL
5. Build Power BI dashboard
6. Generate business insights
