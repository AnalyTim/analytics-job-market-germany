# Project Plan

## Project Goal

Analyze the German job market for Analytics positions and identify the most common roles, skills, locations, employers, salary ranges and working models.

---

## Project Objectives

- Collect job postings from German job platforms
- Clean and standardize the collected data
- Store the cleaned data in PostgreSQL
- Analyze the data using SQL
- Build an interactive Power BI dashboard
- Summarize the most important findings

---

## Planned Data Sources

- StepStone
- LinkedIn Jobs
- Indeed Germany
- Xing Jobs
- Glassdoor

---

## Required Data Fields

- Job title
- Company name
- Location
- Job description
- Date posted
- Employment type
- Remote, hybrid or on-site status
- Salary information
- Job posting URL
- Source platform

---

## Planned Deliverables

- Raw dataset
- Cleaned dataset
- PostgreSQL database
- SQL analysis
- Power BI dashboard
- Project documentation
- Final business insights
